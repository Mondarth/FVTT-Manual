.. dropdown:: Canvas Layer Controls

   These controls apply to all :ref:`canvas` layers.

   .. card::
 
      Keyboard
      ^^^
 
      .. list-table::

         * - :kbd:`alt`
           - Highlight canvas objects
         * - :kbd:`C`
           - Display :doc:`/all/ui/sheets/actor-sheet` of :ref:`player config player character`
         * - :kbd:`tab`
           - Cycle tokens or Reset view.
         * - :kbd:`ctrl` |and| (|up|  | |down| | |left| | |right|)
           - Pan canvas
         * - :kbd:`pageUp` | Numpad :kbd:`+` 
           - Zoom in
         * - :kbd:`pageDown` | Numpad :kbd:`-` 
           - Zoom out

   .. card::

      Mouse
      ^^^

      These are generalised description, individual tools will have more detailed descriptions.

      .. list-table::

         * - |LMB|
           - Select object
         * - :kbd:`shift` |and| |LMB|
           - Toggle object`s selected state
         * - |hLMB| |and| drag
           - Use active tool
         * - |RMB|
           - Cancel active tool operation
         * - |hRMB| |and| drag
           - Pan canvas
         * - Mouse Wheel
           - Zoom canvas
