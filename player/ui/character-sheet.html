Character Sheet

The quickest way to access your character’s details (character sheet) is by pressing C on the keyboard. This will open your primary character sheet [1].

The details for any character you can view can be opened by a LMB on the actor in the Actors tab [2].

Change Character Sheet appearance

There may be more than one Sheet available for character sheets. You can check using the sheet monfiguration button (top right of the character sheet).

Footnotes

[1]

While most players will control a single character you may find yourself running more than one (for example, a magic user may have a familiar that the GM has provided as a controllable actor). When this happens the C opens the primary character’s sheet (set in User Configuration dialog).

[2]

You need to LMB on the text (the mouse cursor changes to the hand), otherwise nothing will happen.
