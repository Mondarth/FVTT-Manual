Players List

Find it

LMB Players

Show me

In the lower left corner of the Foundry UI is the Players list. When collapsed, this shows the currently active players’ user names next to small circles coloured according to the setting in user configuration.

Show me

The list can be expanded to show all players (LMBPlayers) regardless of whether they are currently connected to the game session. The expanded list shows all players registered in the game. Each player has a circle to the left of their name, if this circle is empty the player is not connected to this game session. When a player is connected to the game session the circle is filled with their selected Player Color.

Players Context Menu

Find it

You are ‘Player2’ and are opening the Players context menu.

RMB Player2

Show me

Note

The context menu content will vary.

For example, a GM has access to additional actions and players may be able to view another players avatar (if set) but will not have access to other players User Configuration dialog

Player List Context Menu Options

User Configuration

Opens the User Configuration dialog

View Player Avatar

Displays the avatar set by the player in Player Avatar.

If no avatar is set, this option is not shown.

Pull To Scene

Force the player onto the current scene[2].

Kick Player

Force the player to log out of this game session

Ban Player

Force the player to log out of this game session and

prevent them from rejoining[1].

Footnotes

[1]

Internally this sets the players account role to None. To allow the player back the GM must restore the player’s account to at least the Players role.

[2]

Sometimes players get lost, lag behind as the party move on, or simply ‘wander off’ (especially if there are multiple scenes available)—or the system may get confused. Either way, this function allows the GM to force players onto a scene.
