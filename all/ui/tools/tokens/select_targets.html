Select Targets

Find it

left-click

Show me
In a nutshell

Acts on the Token Actors Layer.

Keyboard

T

Target the token under the mouse cursor.

shift T

Token under the mouse cursor is added to,

or removed from, existing targetting selection.

Mouse

LMB

Target the token under the cursor.

shift LMB

Toggle targeted state of the token.

Hold LMB drag

Target all tokens in the band.

shift Hold LMB drag

Add all tokens in the band to targeted selection.

Attention

Any time the Token Controls tool is active you can target tokens with Double RMB or T (and holding shift toggles the token’s targetted state).

Target individual token

Either,

  1. target the token with LMB or

  2. place mouse cursor over token and press T.

Note

Any existing selection will be cancelled and replaced with this new selection.

Target multiple tokens

To target more than one token at a time, Hold LMB drag. You will see a rectangular band appear as you drag. Any token that falls inside the band will be selected when the LMB is released, any previously targeted tokens are untargeted.

Important

Target tokens do not need to be completely inside the selection band. They are selected if any part of their token is in the band.

Important

Selection is based on the token’s tile, not its image. Some token images are larger than the token tile.

To add multiple targets to selection, hold shift Hold LMB drag.

Add target to selection

To add another target either,

  1. shift LMB over untargeted token or

  2. place mouse cursor over untargeted token and press shift T

Add multiple targets to selection

To add multiple tokens to selected targets, shift Hold LMB drag.

Note

shift does not untarget tokens when using this method.

Untarget individual token

To untarget a token either,

  1. shift LMB on a targeted token, or

  2. place mouse cursor over a targeted token and shift T.
