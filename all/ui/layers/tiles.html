Tiles Layer

The tile layer is edited using Tile Controls .

Tile Types

There are two categories of tile on the tiles layer; standard and overhead. A tile’s elevation dictates which ‘type’ it is considered to be.

Attention

The tiles layer distinguishes these two file types and restricts your ability to interact with them. If you are currently on the ‘background’ then you can interact with standard if you are on the ‘foreground’ you can only interact with overhead tiles.

While editing tiles with the Tile Controls you can swich between editing standard or over head tiles using the Foreground Layer.

Standard Tiles

Also called ‘background’ tiles these are any tile with an elevation less than that set for the foreground image.

Overhead Tiles

The distinction between ‘Sandard Tiles’ and Overhead Tiles is slightly diffferent in FVTT V12. In the past a checkboc indicated when a tile was to be considered ‘overhead’, in FVTT V12 the tile’s tile elevation dictates whether a tile is overhead. Any tiles with an elevation equal to, or higher than, the elevation set for the foreground image is considered to be an overhead tile.
