.. dropdown:: Global Controls
  
   These controls apply whereve
   you are in the Foundry |UI|.

   .. card::
 
      Global Keys
      ^^^
 
      .. list-table::
 
        * - :kbd:`F5` 
          - Refresh browser
        * - :kbd:`ctrl` |and| :kbd:`F5`
          - Refresh without cache. This will force the browser to download `all` the application conten from the server.
        * - :kbd:`esc`
          - Close all currently ope application windows

            If there are no currently open application windows then the :doc:`/all/ui/main-menu` is toggled.
        * - :kbd:`space`
          - Toggle game paused
        * - :kbd:`1` | :kbd:`2` | :kbd:`3` | :kbd:`4` | :kbd:`5` | :kbd:`6` | :kbd:`7` | :kbd:`8` | :kbd:`9` | :kbd:`0`
          - Activate :doc:`/all/ui/hotbar` slot
        * - :kbd:`alt` |and| (:kbd:`1` | :kbd:`2` | :kbd:`3` | :kbd:`4` | :kbd:`5` | :kbd:`6` | :kbd:`7` | :kbd:`8` | :kbd:`9` | :kbd:`0`)
          - Swap to :doc:`/all/ui/hotbar` page by number
        * - :kbd:`shift` |and| :kbd:`C`
          - Switch focus to chat entry on :doc:`/all/ui/tabs/chat`
