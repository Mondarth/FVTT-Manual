Global Controls

These controls apply whereve you are in the Foundry UI.

Global Keys

F5

Refresh browser

ctrl F5

Refresh without cache. This will force the browser to download all the application conten from the server.

esc

Close all currently ope application windows

If there are no currently open application windows then the Main Menu is toggled.

space

Toggle game paused

1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 | 9 | 0

Activate Hotbar Area slot

alt (1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 | 9 | 0)

Swap to Hotbar Area page by number

shift C

Switch focus to chat entry on Chat Messages Tab
