Canvas Layer Controls

These controls apply to all Canvas layers.

Keyboard

alt

Highlight canvas objects

C

Display Actor Sheet of Player Character

tab

Cycle tokens or Reset view.

ctrl ( | | | )

Pan canvas

pageUp | Numpad +

Zoom in

pageDown | Numpad -

Zoom out

Mouse

shift LMB
